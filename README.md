Настольное приложение Scheduler [2026.01.08] было разработано для помощи учащимся в школе.
Программа направлена на работу с операционной системой Windows (протестировано на версиях 10 и 11 [64-бит]),
однако она была протестирована и на системе с ядром Linux (Debian).

Приложение позволяет просматривать расписания уроков, видеть, какие уроки уже закончились,
какие идут прямо сейчас и какие скоро будут, создавать и изменять расписания, 
просматривать аналитику по предметам, поддерживает кастомизацию внешнего вида
(позволяет устанавливать свои шрифты, выбирать размер текста в интерфейсе,
тему (палитру) интерфейса и загрузочную анимацию), а также может проверить целостность
своих файлов и составить интерактивный план по сборке рюкзака на выбранный день:
что нужно из него убрать, что добавить и что в итоге в нём должно быть.
Программа была разработана с целью помощи учащимся с тем,
чтобы правильно собирать портфель на предстоящие занятия и ничего не забывать!

Для установки приложения достаточно извлечь содержимое архива программы в пустую папку.
Чтобы запустить приложение, откройте файл Scheduler.exe и дождитесь загрузки.
